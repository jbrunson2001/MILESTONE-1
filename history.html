History of Browsers

The first web browser was created by Tim Berners-Lee. Early web browsers were used mostly to view simple documents and links.

Browsers such as Mosaic and Netscape helped make the web more popular by making pages easier to see. For many years, Internet Explorer was the most widely used. Later on, Firefox, Chrome, Safari, and Edge also became major browsers.

Timeline

  • 1990: First web browser
  • 1993: Mosaic helped make the web popular
  • 1990s: Netscape and Internet Explorer competed
  • 2000s: Firefox and Chrome became important
  • Today: Chrome, Safari, Edge, Firefox, and others are common
