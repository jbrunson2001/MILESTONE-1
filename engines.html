Browser Engines

A browser engine is the part of a browser that shows you the web page. It reads the HTML, adds the CSS styles, and handles JavaScript

Blink

Used by Chrome, Edge, Brave, and Opera

WebKit

Used by Safari

Gecko

Used by Firefox

Why it matters

Various engines make sure the web stays open and works well across different browsers
